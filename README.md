About

This repository contains solutions to various coding challenges. Problems are organized by platform and difficulty, aiming to help anyone learn algorithms, data structures, and problem-solving techniques in JavaScript.
Platforms
LeetCode – popular platform for interview-style problems
Codewars – fun and challenging coding exercises
Languages
JavaScript – all solutions are written in JavaScript (ES6+)
Folder Structure
CodeHub/
├─ LeetCode/
│  ├─ Easy/
│  ├─ Medium/
│  └─ Hard/
├─ Codewars/
│  ├─ 8 kyu/
│  ├─ 7 kyu/
│  └─ 6 kyu/
└─ README.md
Each file is named after the problem it solves, e.g., twoSum.js.
How to Contribute
Fork the repository.
Create a new branch: git checkout -b feature/problem-name.
Add your solution in the appropriate folder.
Commit your changes: git commit -m "Add solution for Problem Name".
Push to your branch: git push origin feature/problem-name.
Open a Pull Request.

Make sure your code is clean, commented, and working before submitting.
